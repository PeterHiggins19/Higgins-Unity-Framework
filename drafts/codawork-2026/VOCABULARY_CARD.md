## SAY THIS (CoDa vocabulary)

| CoDa Term | Meaning | Instead of saying... |
|-----------|---------|---------------------|
| Composition | A vector of proportions that sum to a constant | "share vector", "mix", "allocation" |
| Component | One element of the composition | "carrier", "share", "slice" |
| Simplex (S^D) | The space where compositions live | "unit space", "proportion space" |
| Perturbation | Compositional change (multiplicative) | "drift", "shift", "change" |
| Closure | The constant-sum constraint | "unity constraint", "budget constraint" |
| Aitchison distance | The natural metric on the simplex | "distance", "difference" |
| Log-ratio | Logarithm of a ratio between components | "log difference", "dB" |
| CLR transform | Centered log-ratio: ln(xᵢ/geometric mean) | "normalization" |
| ILR transform | Isometric log-ratio: orthonormal coordinates | "projection", "decomposition" |
| Balance | ILR coordinate from a binary partition | "contrast", "crossover" |
| Sub-composition | A subset of components, re-closed | "sub-mix", "partial allocation" |
| Structural zero | True absence of a component | "empty carrier", "dead channel" |
| Rounded zero | Below detection, not truly absent | "near-zero", "trace" |
| Aitchison geometry | The vector space structure of the simplex | "simplex math" |
| Aitchison norm | Distance from uniform composition | "concentration", "peakedness" |
| Sequential binary partition (SBP) | Tree that defines balances | "hierarchy", "crossover network" |
| Subcompositional coherence | Results hold on any subset of components | "consistency across subsets" |
| Compositional biplot | PCA on CLR-transformed data | "spectrum display" |

## PHRASES THAT LAND WELL

| Say this | Why it works |
|----------|-------------|
| "The geometry is yours." | Acknowledges 40 years of CoDa work in four words. |
| "I'm not claiming new mathematics." | Disarms the "what's new?" attack instantly. |
| "I'd welcome guidance on..." | Turns a gap into a collaboration offer. |
| "That's one of my open gaps." | Honest. Specific. Respected. |
| "The data is public." | Signals transparency and invites replication. |
| "Four specific ways to defeat the claim." | Scientists respect falsifiability above all else. |
| "In your terms, that would be..." | Shows you learned their language. |
| "I don't know. How would you approach it?" | The most respected sentence at any conference. |

---

## PHRASES THAT WILL HURT YOU

| Never say | Why it's dangerous |
|-----------|-------------------|
| "CoDa is philosophizing" | Calls their life's work unrigorous. Unrecoverable insult. |
| "The math is obvious" | Implies their 40 years was trivial. |
| "In the real world..." | Implies their work isn't real. Every geologist in the room has done fieldwork. |
| "My framework is better because..." | Competitive framing. You're there to collaborate, not compete. |
| "HUF solves this" | Premature claim. HUF proposes a monitoring approach. It hasn't been validated yet. |
| "I already thought of that" | Even if true, it sounds dismissive. Say "Yes — that's exactly where I need help." |
| "The AI told me..." | Never lead with AI. If asked, disclose. Never volunteer as a credential. |
