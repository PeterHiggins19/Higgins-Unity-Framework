# 🏙️ **Complete List of Scrapable Data Sources for Markham Urban Planners**

## 🟦 **1. Official City of Markham Open Data Sources**  
These are the most authoritative and fully scrapable datasets.

### **1.1 Markham Open Data Portal (ArcGIS Hub)**  
**URL:** https://data-markham.opendata.arcgis.com/

**Scrapable datasets include:**
- Road network  
- Trails  
- Bicycle routes  
- Address points  
- Parcels (if published)  
- Parks & recreation assets  
- Stormwater assets  
- Streetlights  
- Trees & canopy  
- Heritage properties  
- Zoning layers (if published)  
- Development applications (if published)  
- Ward boundaries  
- Transit routes (if provided by YRT/Viva)  
- Floodplain & environmental layers (if published)

**Scraping methods:**  
- ArcGIS REST API  
- GeoJSON  
- FeatureServer endpoints  
- CSV/SHP downloads  
- Automated ETL via Python (requests + arcgis)
