# ⚠️ 4. Limitations for Urban Planning

### **Not a Design or Modeling Tool**  
POSSE does **not** handle:  
- 3D modeling  
- Urban design visualization  
- Scenario planning  
- Land‑use forecasting  
- Transportation modeling  

You would still need tools like:  
- ArcGIS Urban  
- Envision Tomorrow  
- UrbanFootprint  
- CityEngine  
- EMME / VISUM  
- QGIS + plugins

### **Requires Configuration Effort**  
Because POSSE is a platform, not a turnkey planning suite, cities must invest in:  
- Workflow design  
- Data integration  
- Change management  
- Staff training

### **Not a Public Engagement Platform**  
It supports portals, but not deliberative engagement (e.g., surveys, mapping feedback, participatory planning).
