# Compositional Monitoring Protocol

Perturbation-based drift detection on the simplex using Aitchison geometry.

**Protocol:** Δ(t) = C(x₁(t+1)/x₁(t), …, xD(t+1)/xD(t)), distance d_A = ‖clr(x) − clr(y)‖, effective diversity N_eff = exp(H), drift flagged at μ+2σ.

**Falsifiable claim:** No existing monitoring framework performs compositional change detection at the carrier level with formal perturbation-based drift measurement.

- UK: 2004-2005 d_A=2.98, 2017-2018 d_A=3.23, 2019-2020 d_A=3.26
