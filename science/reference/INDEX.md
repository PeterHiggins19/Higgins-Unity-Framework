# HUF Reference Collection

**The core documents, in learning order.**

Each document is self-contained. Read in order for the full picture, or jump to any one for a specific topic. Documents marked with a source pointer trace back to the definitive governance record.

## The Learning Path

| # | Document | Time | What You Learn |
|---|----------|------|----------------|
| 01 | [Executive Summary](01_EXECUTIVE_SUMMARY.md) | 5 min | The cylinder problem, the fuel gauge metaphor, one-sentence HUF |
| 02 | [The Four Monitoring Categories](02_THE_FOUR_MONITORING_CATEGORIES.md) | 10 min | MC-1 through MC-4, why MC-4 was invisible, academic lineage |

| 03b | [The Ontological Foundation](03b_THE_ONTOLOGICAL_FOUNDATION.md) | 10 min | The pre-existing condition, the invisible door, the zero state |
| 04 | [EITT Complete Explanation](04_EITT_COMPLETE_EXPLANATION.md) | 15 min | The discovery narrative, what it means, what remains open |
| 05 | [EITT Finding](05_EITT_FINDING.md) | 10 min | The empirical results, the numbers, cross-domain confirmation |
| 06 | [EITT CoDa Mathematics](06_EITT_CODA_MATHEMATICS.md) | 20 min | Formal CoDa backbone, Aitchison geometry, proofs and derivations |
