HUF CoDa Explorer — EMBER Energy Compositions

Six visualization methods for compositional drift detection · Germany / Japan / UK · 2000–2025
Germany Japan United Kingdom
1. Ternary Simplex (S²)
Sub-composition: Fossil / Nuclear / Renewable. Each point = one year. The 2-simplex — geology's ternary diagram applied to energy.
CoDa method: Sub-compositional projection onto S². Closure C(fossil, nuclear, renewable) = 1. This is the view geologists use for sand/silt/clay.
2. Simplex Time Prism ("Toblerone")
The ternary triangle extruded through time. Each horizontal slice is one year's composition. The trajectory becomes a ribbon in 3D space.
CoDa method: Compositional time series as a path on the simplex. The 3rd axis (depth) = time. Structural shocks appear as sharp bends in the ribbon.
3. CLR-Transformed Carriers
Centered log-ratio: ln(xᵢ / geometric mean). Transforms the simplex to real space. Values above 0 = above-average share. Below 0 = below average.
CoDa method: CLR transform — the standard CoDa visualization. Removes closure constraint so standard statistics apply. Crossing zero = carrier share crosses the geometric mean.
4. Perturbation Velocity
Year-over-year compositional "difference": p(t) ⊖ p(t−1) = C(xᵢ(t)/xᵢ(t−1)). The simplex equivalent of a first derivative. Spikes = structural shocks.
CoDa method: Perturbation difference — the "subtraction" on the simplex. Aitchison distance between consecutive years measures total compositional velocity. Fukushima and UK coal exit appear as spikes.
5. Balance Time Series
Hierarchical log-ratio balances. B1 = ln(Fossil/Renewable) — the energy transition balance. B2 = ln(Coal/Gas) within fossil. B3 = ln(Wind/Solar) within renewable.
CoDa method: ILR balances via sequential binary partition. Each balance is a single interpretable log-ratio between groups. B1 crossing zero = renewables overtake fossil.
6. Aitchison Distance Matrix
Pairwise Aitchison distance between every pair of years. Bright = large compositional change. Diagonal = zero. Off-diagonal blocks reveal regime boundaries.
CoDa method: Aitchison distance d_A(x,y) — the natural metric on the simplex. Computed on full 9-carrier compositions (no sub-composition). Block structure reveals regime transitions.
