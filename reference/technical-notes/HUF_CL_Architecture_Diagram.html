CL-02
Scale-Invariant Drift Trigger
FM Precedent: Limiter + Capture-Effect Discriminator
ADD
Computes Total Variation distance TV(ρₜ, ρₜ₋₁) = ½Σ|ρᵢ − ρᵢ'| and applies drift threshold. Scale-blind: responds to proportional change, not absolute magnitude. Distinguishes pulse from drift.

CL-03
Effective Complexity Gauge
FM Precedent: FM Discriminator S-Curve
ADD*
Computes Keff = exp(−Σρᵢ ln ρᵢ) and complexity retention ratio χₜ = Keff,ₜ / Keff*. Detects deceptive drift: structural concentration invisible to velocity monitoring. S-curve confirmed by simulation.
Output → Keff value, χₜ ratio, structural health assessment

Gain-Scheduled Verdict Logic

ADD

Output → Governance verdict:
● GREEN — Stasis
● YELLOW — Permissive Drift
● RED — Critical Breach
