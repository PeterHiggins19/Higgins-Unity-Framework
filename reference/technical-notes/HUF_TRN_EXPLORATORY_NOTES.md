## Core Concept

Any system whose observables can be expressed as proportions summing to a whole
lives on the simplex. The entire CoDa/EITT toolkit therefore applies in principle
to any domain where ratios-of-parts are the natural measurement.

HUF-TRN ("Inter-Systems Transformers") names this observation explicitly:
transformer ratios between systems provide a unified ratio-metric interface
for inter-system diagnostics across time, space, mass, and energy.

**What this is**: A recognition that compositional analysis is domain-agnostic
by construction. Anything expressible as proportions → simplex → ILR → EITT.

**What this is not**: A new discovery. This follows directly from the definition
of the simplex. The value is in naming it and asking: where else does it apply?

## Pairwise Simplex Ratios ("Turns Ratios")

Peter's further idea (2026-04-10): assign a transformer coupling (turns ratio)
between every pair of elements on the simplex. Each pairwise log-ratio
ln(x_i / x_j) represents the "exchange rate" between parts i and j.

**CoDa connection**: This is exactly what CoDa already does. The pairwise
log-ratios are the fundamental atoms of compositional data analysis:

- **ALR** (Additive Log-Ratio): all ratios relative to one chosen reference part
- **CLR** (Centered Log-Ratio): all ratios relative to the geometric mean
- **ILR** (Isometric Log-Ratio): orthogonal contrasts built from pairwise ratios
  via a Sequential Binary Partition

The "turns ratio" intuition maps naturally to ILR balances: each balance IS a
weighted geometric-mean ratio between two groups of parts. The SBP tree defines
which parts couple to which, and the ILR coefficients are the turns ratios.

**What's potentially new**: The idea of treating inter-SYSTEM ratios (not just
intra-system part ratios) as compositional. E.g., if System A has K₁ parts and
System B has K₂ parts, can you form a meta-simplex of (K₁ + K₂) parts and
compute cross-system ILR balances? This is conceptually valid if the parts share
a common unit (e.g., energy fractions across two grids). It is NOT valid if the
parts measure incommensurable quantities.
